# Bintou Website (Static)

This is a fully static webpage built from the provided Figma design.

## Run locally

Open `index.html` in your browser, or run:

```bash
python3 -m http.server 8080
```

Then open:

- `http://localhost:8080`

## Assets

Images are stored in `./assets/images/` (downloaded from the Figma file).
